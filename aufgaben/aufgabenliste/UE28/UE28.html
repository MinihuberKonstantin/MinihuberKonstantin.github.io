Einige kleine Flexbox - Fragen

Welchem grundsätzlichem Bereich / Element muss im CSS die Eigenschaft DISPLAY: FLEX zugewiesen werden?

Einen div-Tag.

Wo liegt der Unterschied zwischen den Werten JUSTIFY-CONTENT und ALIGN-ITEMS?

Das eine zentriert horizontal und das andere vertikal.

Nenne mind. 3 Einstellungsmöglichkeiten für den Wert JUSTIFY-CONTENT.

baseline, center, end.

Nenne mind. 3 Einstellungsmöglichkeiten für den Wert ALIGN-ITEMS.

baseline, center, flex-end.

Welche 2 Achsen beeinflussen eure Flexboxen im Hintergrund?

width, height?

Verhalten sich diese 2 Achsen immer gleich, oder gibt es Situationen wo sie sich ändern?

Wenn man sie mit CSS umstellt.

Wie ist mittels Flexboxen ein "perfektes Zentrieren" möglich?

Mit justify-content: center, und align-items: center.
